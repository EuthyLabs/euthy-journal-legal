Kebijakan Privasi

Janji Privasi Kami

Pikiran, gambar, dan memo suara Anda adalah milik Anda sendiri. Euthy Journal dibuat dan dioperasikan oleh Yhonathon Maharaj dan dibangun dengan filosofi "Local-First". Kami percaya jurnal pribadi Anda tidak boleh disimpan di server pengembang di mana ia bisa diakses, dipindai, atau dibocorkan.

1. Penyimpanan Data & Cadangan Cloud

Entri jurnal, foto, gambar, catatan suara, rutinitas, tag, data suasana hati, e dan cuplikan kesehatan yang disimpan disimpan secara pribadi di perangkat Anda. Untuk melindungi dari kehilangan data, aplikasi menawarkan cadangan cloud opsional. Jika diaktifkan, salinan data Anda dienkripsi dan disimpan di dalam akun Apple iCloud (iOS) atau Google Drive (Android) pribadi Anda. Euthy Journal menggunakan cakupan 'Data Aplikasi' yang dibatasi; aplikasi hanya dapat mengakses folder dan file tertentu yang dibuatnya dan tidak memiliki izin untuk melihat, membaca, atau memodifikasi konten lain di penyimpanan cloud pribadi Anda. Kami tidak mengoperasikan database pusat kami sendiri dan tidak dapat mengakses, membaca, atau membagikan konten ini.

2. Cadangan Otomatis

Backup otomatis secara berkala membuat arsip ZIP dari jurnal Anda dan mengunggahnya ke folder yang terlihat di iCloud Drive atau Google Drive pribadi Anda. File-file ini dikendalikan sepenuhnya oleh Anda dan tunduk pada ketentuan serta keamanan penyedia cloud Anda.

3. Keamanan & Biometrik

Jika Anda mengaktifkan kunci PIN atau Biometrik (FaceID/TouchID/Sidik Jari), aplikasi menggunakan sistem keamanan asli perangkat Anda. PIN Anda disimpan dengan aman di perangkat Anda, dan data biometrik Anda tidak pernah dibagikan dengan aplikasi atau pihak ketiga mana pun. Backup cloud Anda dilindungi oleh keamanan akun Apple atau Google Anda.

4. Integrasi Kesehatan (Apple Health & Health Connect)

Jika Anda memilih untuk mengaktifkan integrasi kesehatan, aplikasi akan membaca metrik tertentu termasuk langkah kaki, tidur, detak jantung, dan latihan/olahraga untuk memberikan wawasan tentang bagaimana aktivitas Anda mempengaruhi suasana hati Anda. Data ini dibaca langsung dari penyimpanan kesehatan perangkat Anda dan disimpan di dalam entri jurnal Anda hanya saat Anda menyimpannya. Kami tidak pernah mengunggah data kesehatan Anda ke server kami, dan kami tidak membagikannya dengan pihak ketiga atau menggunakannya untuk iklan.

Retensi & Penghapusan: Data kesehatan hanya disimpan selama entri jurnal yang melampirkannya masih ada. Menghapus entri jurnal tertentu akan menghapus data kesehatan terkait secara permanen. Menghapus aplikasi Euthy Journal dari perangkat Anda akan menghapus semua data yang disimpan secara lokal.

5. Data Lokasi & Cuaca

Jika Anda memilih untuk menggunakan fitur penandaan lokasi, peta, atau cuaca, aplikasi akan meminta akses ke layanan lokasi perangkat Anda. Koordinat lokasi tepat, tempat yang ditandai, dan kondisi cuaca yang diambil disimpan secara lokal di perangkat Anda di dalam entri jurnal Anda. Untuk menyediakan pencarian alamat (geocoding), tampilan peta, dan kondisi cuaca saat ini, koordinat Anda diproses oleh layanan asli (Apple Maps dan WeatherKit di iOS, Google Maps dan Google Weather API di Android). Ini hanya terjadi saat Anda secara eksplisit memicu fitur-fitur ini (misalnya, dengan mengeklik tombol lokasi atau cuaca). Kami tidak menyimpan data lokasi atau cuaca Anda di server kami atau melacak pergerakan Anda di latar belakang.

6. Statistik & Analisis Di Perangkat

Semua statistik, pola, dan ringkasan (seperti "Persona Jurnal" atau "Ringkasan Mingguan" Anda) dihasilkan secara ketat di perangkat Anda menggunakan algoritma lokal. Teks jurnal dan data aktivitas Anda tidak pernah dikirim ke server eksternal atau layanan AI pihak ketiga untuk analisis. Semua refleksi dan pemrosesan terjadi sepenuhnya di dalam aplikasi Euthy Journal di ponsel Anda.

7. Performa & Laporan Crash

Untuk memastikan aplikasi stabil dan berkinerja baik, kami mungkin menggunakan layanan analitik, pemantauan performa, dan pelaporan crash. Data ini digunakan untuk mengidentifikasi bug teknis dan mengoptimalkan keandalan. Ini mungkin mencakup informasi teknis seperti versi aplikasi, model perangkat, versi sistem operasi, dan log crash. Ini tidak mencakup entri jurnal, foto, gambar tangan, catatan suara, atau konten jurnal pribadi lainnya.

8. Hak Uni Eropa (GDPR)

Jika Anda adalah penduduk Uni Eropa, Anda memiliki hak-hak tertentu di bawah General Data Protection Regulation (GDPR):

  • Hak Akses: Anda dapat mengekspor data jurnal inti Anda (teks, suasana hati, tag) kapan saja secara gratis melalui alat Ekspor JSON di Pengaturan.
  • Hak Penghapusan: Anda dapat menghapus semua data Anda secara permanen menggunakan tombol 'Hapus Semua Data' di Pengaturan.
  • Hak Portabilitas: Data Anda diekspor dalam format standar (JSON) untuk digunakan di layanan lain. Format lanjutan (PDF, ZIP) tersedia melalui Premium.
  • Hak Keberatan: Anda dapat memilih keluar dari analitik penggunaan anonim dan pemantauan teknis kapan saja melalui toggle di Pengaturan.

Untuk menggunakan Hak Penghapusan Anda atas data yang diproses oleh sub-prosesor pihak ketiga kami (Sentry dan RevenueCat), silakan hubungi kami di: euthylabs@gmail.com dengan ID Dukungan Anda, yang dapat ditemukan di bagian Pengaturan > Tentang di aplikasi.

9. Sub-Prosesor Data

Untuk menyediakan layanan kami, kami menggunakan sejumlah terbatas sub-prosesor pihak ketiga yang tepercaya :

  • Aptabase: Menyediakan analitik penggunaan anonim yang mengutamakan privasi.
  • Sentry: Menyediakan pelaporan crash teknis dan pemantauan performa untuk memastikan stabilitas aplikasi.
  • RevenueCat: Mengelola status langganan dan verifikasi pembelian.
  • Apple (iCloud & WeatherKit) / Google (Drive & Weather API): Menangani backup cloud opsional Anda dan mengambil data cuaca saat diminta. Kami tidak pernah memiliki akses ke kredensial akun cloud Anda.

10. Langganan

Pembayaran langganan ditangani dengan aman oleh Apple dan Google. Kami menggunakan RevenueCat untuk mengelola status langganan dan memverifikasi pembelian. RevenueCat bertindak sebagai penyedia layanan dan memproses hanya informasi yang diperlukan untuk memvalidasi langganan Anda, seperti ID transaksi dan pengenal perangkat. Mereka tidak memiliki akses ke entri jurnal pribadi Anda.

11. Nama Impor Pihak Ketiga

Euthy Journal mungkin merujuk ke aplikasi jurnal pihak ketiga hanya untuk mendeskripsikan format ekspor yang kompatibel. Day One, Journey, Daylio, Diarly, Diarium, dan Stoic adalah merek dagang dari pemiliknya masing-masing. Euthy Journal tidak berafiliasi dengan atau didukung oleh mereka.

12. Hubungi Kami

Jika Anda memiliki pertanyaan tentang Kebijakan Privasi ini atau bagaimana data Anda ditangani, silakan hubungi kami di: euthylabs@gmail.com

Ketentuan Layanan

1. Penerimaan Ketentuan

Dengan menggunakan Euthy Journal, Anda menyetujui ketentuan ini. Euthy Journal dibuat dan dioperasikan oleh Yhonathon Maharaj. Jika Anda tidak setuju, mohon untuk tidak menggunakan aplikasi ini.

2. Persyaratan Usia

Anda harus berusia minimal 16 tahun untuk menggunakan Euthy Journal. Dengan menggunakan aplikasi ini, Anda menyatakan bahwa Anda memenuhi persyaratan usia ini. Jika Anda berusia di bawah 16 tahun, Anda hanya boleh menggunakan aplikasi ini dengan keterlibatan dan persetujuan orang tua atau wali sah.

3. Kepemilikan Konten

Anda memegang 100% kepemilikan atas semua yang Anda tulis, rekam, atau unggah ke Euthy Journal. Kami tidak mengklaim hak apa pun atas data Anda. Data jurnal Anda disimpan secara pribadi di perangkat Anda dan, jika Anda memilih untuk mengaktifkan Backup Cloud Otomatis, di akun Apple iCloud atau Google Drive pribadi Anda. Data tersebut tidak pernah disimpan di server kami.

4. Tanggung Jawab & Kehilangan Data

Euthy Journal adalah aplikasi local-first. Ini berarti data jurnal Anda terutama berada di perangkat Anda dan mungkin juga ada di arsip cadangan di dalam akun cloud pribadi Anda. Kami tidak bertanggung jawab atas kehilangan data yang disebabkan oleh kegagalan perangkat, penghapusan aplikasi, kehilangan perangkat Anda, pelanggaran keamanan akun, kegagalan cadangan, file yang rusak, batas penyimpanan cloud pribadi, atau masalah konektivitas. Merupakan tanggung jawab Anda untuk mengelola akun cloud dan keamanan Anda sendiri.

5. Bukan Saran Medis

Euthy Journal menyediakan alat untuk refleksi diri dan wawasan pribadi. Ini BUKAN perangkat medis dan TIDAK dimaksudkan untuk mendiagnosis, mengobati, menyembuhkan, atau mencegah kondisi kesehatan mental atau fisik apa pun. Fitur 'Statistik', 'Sinyal Pemulihan', 'Korelasi Kesehatan', dan 'Pelacakan Suasana Hati' hanya untuk tujuan informasi. Selalu cari saran dari dokter atau penyedia layanan kesehatan berkualifikasi lainnya untuk pertanyaan apa pun yang Anda miliki mengenai kondisi medis. JANGAN PERNAH MENGABAIKAN SARAN MEDIS PROFESIONAL ATAU MENUNDA MENCARINYA KARENA SESUATU YANG ANDA BACA DI APLIKASI INI.

Euthy Journal bukan layanan krisis. Jika Anda mengalami keadaan darurat kesehatan mental, segera hubungi layanan darurat setempat atau hotline krisis.

6. Impor, Ekspor & Layanan Pihak Ketiga

Euthy Journal dapat mengimpor file cadangan dan ekspor yang didukung dari Euthy Journal dan aplikasi jurnal pihak ketiga tertentu. Impor dapat menimpa data yang sudah ada, jadi Anda harus membuat cadangan terbaru sebelum mengimpor. Nama aplikasi pihak ketiga hanya digunakan untuk mendeskripsikan format ekspor yang kompatibel. Day One, Journey, Daylio, Diarly, Diarium, dan Stoic adalah merek dagang dari pemiliknya masing-masing. Euthy Journal tidak berafiliasi dengan atau didukung oleh mereka.

7. Langganan Premium

Fitur premium ditagih berdasarkan langganan. Pembayaran ditangani oleh Apple App Store atau Google Play Store. Anda dapat mengelola atau membatalkan langganan Anda kapan saja melalui pengaturan toko aplikasi perangkat Anda.

8. Batasan Tanggung Jawab

Sejauh diizinkan oleh hukum, Euthy Journal disediakan "apa adanya" dan "sebagaimana tersedia" tanpa jaminan dalam bentuk apa pun. Kami tidak bertanggung jawab atas kerusakan tidak langsung, insidental, khusus, konsekuensial, atau hukuman, termasuk kehilangan entri jurnal, kehilangan cadangan, kerusakan data, atau kehilangan akses ke aplikasi.

9. Analitik & Optimalisasi

Kami mungkin menggunakan analitik, pemantauan performa, dan pelaporan crash untuk meningkatkan stabilitas dan performa aplikasi. Layanan ini mungkin mengumpulkan diagnostik teknis dan pola penggunaan anonim seperti versi aplikasi, model perangkat, versi sistem operasi, log crash, dan interaksi fitur (misalnya tampilan peta, pilihan tema). Mereka tidak mengumpulkan entri jurnal pribadi Anda, foto, gambar tangan, catatan suara, rutinitas, snapshot kesehatan yang disimpan, atau konten jurnal pribadi lainnya.

10. Backend, API & Fitur Cloud

Kami mungkin menambahkan fitur backend, akun, API, atau fitur cloud opsional di masa mendatang. Layanan tersebut dapat digunakan untuk menyediakan fungsionalitas aplikasi yang diminta, dukungan langganan, sinkronisasi, atau fitur terkait akun. Kami tidak akan menggunakan layanan backend untuk menyimpan atau menganalisis konten jurnal pribadi Anda kecuali fitur tersebut secara jelas meminta izin Anda dan Anda memilih untuk mengaktifkannya.

11. Integrasi Cloud

Saat menggunakan Google Drive atau iCloud untuk cadangan, Anda tunduk pada ketentuan masing-masing. Kami hanya meminta izin minimum yang diperlukan untuk menyimpan dan mengambil cadangan jurnal Anda. Euthy Journal menggunakan penyimpanan khusus aplikasi yang dibatasi; kami tidak memiliki akses ke kredensial akun Anda atau data apa pun di luar file cadangan Euthy Journal.

12. Layanan Peta dan Lokasi Pihak Ketiga

Euthy Journal menggunakan layanan peta asli (seperti Apple Maps atau Google Maps) untuk menyediakan fitur penandaan lokasi dan tampilan peta. Dengan menggunakan fitur-fitur ini, Anda juga setuju untuk terikat oleh Ketentuan Layanan masing-masing dari penyedia peta pihak ketiga tersebut. Euthy Journal tidak bertanggung jawab atas keakuratan data geocoding atau tampilan peta yang disediakan oleh pihak ketiga tersebut.

13. Perubahan Ketentuan

Kami dapat memperbarui ketentuan ini dari waktu ke waktu. Penggunaan aplikasi Anda yang berkelanjutan setelah perubahan dilakukan merupakan penerimaan Anda terhadap ketentuan baru.
